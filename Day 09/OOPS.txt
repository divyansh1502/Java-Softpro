OOPS:- OOPS stands for Object Oriented Programming System. It is a mechanism of software development. 

OOPS has 4 pillars:-

1. Abstraction
2. Incapsulation
3. Inheritance
4. Polymorphism

Any programming language which folows these 4 concept is kn ow ans OOP language.

Class:- Class is the blueprint/structure of Object. (OR) A class is a collection of variables and method.

Objects:- Object is a real world entity which contain its own properties and functionalities.

1. Abstraction:- Abstraction is a mechanism to show essential functionalities and hide internal functionalities of an object. 

2. Encapsulation:- Encapsulation is a mechanism to wrap properties and functionalities of object in a single unit.

3. Inheritance:- InInheritance is a feature of object-oriented programming.In Inheritance, you can create a new product by using an existing product. 

4. Polymorphism:- The term polymorphism means One thing, many forms.
